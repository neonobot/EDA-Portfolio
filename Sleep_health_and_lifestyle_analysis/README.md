## **Анализ факторов влияющих на наличие расстройств сна**

## Цель
Выявить факторы влияющие на наличие расстройств сна.

## Данные
**Источник:** [Kaggle - Lifestyle and Sleep Patterns](https://www.kaggle.com/datasets/minahilfatima12328/lifestyle-and-sleep-patterns)

Этот набор данных демонстрирует взаимосвязь между образом жизни и его влиянием на общее состояние сна. Он включает подробную информацию о физиологических факторах человека, образе жизни и состоянии сна, в том числе об артериальном давлении, частоте сердечных сокращений, физической активности, индексе массы тела (ИМТ), количестве шагов в день, уровне стресса, продолжительности сна, качестве сна и наличии нарушений сна.



| Признак | Тип | Описание |
|---------|-----|----------|
| `Person ID	` | int | Идентификатор человека |
| `Gender` | object | Гендер |
| `Age` | int | Возраст |
| `Occupation` | object | Род деятельности |
| `Sleep Duration` | float | Продолжительность сна |
| `Quality of Sleep` | int | Качество сна |
| `Physical Activity Level` | int | Уровень физической активности |
| `Stress Level` | int | Уровень стресса |
| `BMI Category` | object | Весовая категория |
| `Blood Pressure` | object | Артериальное давление |
| `Heart Rate` | int | Частота сердечных сокращений |
| `Daily Steps` | int | Кол-во шагов в день |
| `Sleep Disorder` | object | Расстройства сна |

## Методология
1. **Первичный осмотр:** Размер данных, типы, пропуски, дубликаты
2. **Анализ распределений:** Распределение числовых и категориальных признаков
3. **Обработка данных:** Обработка пропусков, преобразование категориальных признаков, удаление дубликатов
4. **Статистические тесты:** Проверка гипотез

## Подтвержденные гипотезы:
1. Наличие расстройств сна влияет на его продолжительность и качество.
2. ИМТ является важным фактором, влияющим на наличие расстройств сна.
